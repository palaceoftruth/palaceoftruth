# Code of Conduct

Palace of Truth follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Report conduct issues through the public repository's maintainer contact path. If
the report includes private user content, secrets, or security-sensitive
operational details, use GitHub private vulnerability reporting instead of a
public issue.

Maintainers may remove comments, issues, pull requests, or other contributions
that violate the code of conduct. Repeated or severe violations may result in a
temporary or permanent ban from project spaces.
