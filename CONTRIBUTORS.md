# Contributors

This file is kept as a compatibility alias for older links. The current
contributor guide is [CONTRIBUTING.md](CONTRIBUTING.md); prefer that file for
development workflow, PR expectations, and verification commands.

Palace of Truth uses Git history and GitHub pull request attribution as the
source of truth for individual contributions. Add explicit acknowledgments here
only when the project adopts a contributor recognition policy.

Historical workflow notes were removed from this compatibility file; use git history if you need older private workflow context.
